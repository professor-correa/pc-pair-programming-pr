## Componente criado
Criado para conquistar o achievement de Co-author

- Tela Inicial com nossos nomes, botão de boas-vindas e fundo em movimento
- Oque foi utilizado: 
    
    - < link > -> Direciona para o google fonts
    - < main > -> Agrupa todo o conteúdo da tela
    - < video > -> Mostra um vídeo sendo o background
    - < classe conteudo > -> section que agrupa o h1 e o botão
