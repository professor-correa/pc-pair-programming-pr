# Botão Fugitivo

Um pequeno experimento em HTML, CSS e JavaScript que testa sua paciência e reflexos: um botão que **foge do cursor do mouse**! E se você conseguir pegar ele... ele fica um pouquinho maior!

## Funcionalidades

- Quando o cursor se aproxima, o botão foge para outra posição na tela.
- Se o usuário conseguir pegar, o botão aumenta de tamanho.
- Estilização divertida e responsiva.

## Tecnologias

- HTML
- CSS
- JavaScript puro (sem frameworks!)
