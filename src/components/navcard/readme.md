# Projeto em Bootstrap - navbar e Card

Este é um projeto básico criado para testar o framework bootstrap e as boas práticas. Usamos coisas simples apra simular uma interface mais simplória, apenas com uma navbar e um card.

## Tecnologias utilizadas
- HTML
- CSS
- Bootstrap

## Estrutura do Projeto
O projeto é composto por:

Navbar: posicionada no topo da página, responsiva e com links de navegação.

Card: componente com imagem, título, texto e botão.

## Como usar
Clone ou baixe este repositório:

- bash
- Copy
- Edit

```git clone https://github.com/seu-usuario/seu-repositorio.git```

Abra o arquivo index.html em qualquer navegador moderno.

## Exemplo de uso
A navbar pode ser usada como menu principal de um site, enquanto o card serve como um destaque de conteúdo, produto ou serviço.

## Classes utilizadas para estilização

- tag a
- class .dropdown-menu > li > a


## Observações:

O projeto utiliza apenas recursos nativos do Bootstrap, sem necessidade de JavaScript personalizado.

O layout é totalmente responsivo.
